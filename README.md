# Lupanulla • Jukwaa la Elimu Tanzania

**Jukwaa kuu la elimu Tanzania** — Darasa la 1 hadi Chuo Kikuu.  
**Notes, Past Papers, Kamusi, Kikokotoo, Video, Blog na AI** — yote bure!

![Lupanulla](https://Lupanulla.co.tz)

## 🌟 Sifa Kuu (Features)

- **📚 Notes & Mada** — Darasa la Msingi, Sekondari na Vyuo
- **📄 Past Papers** — NECTA (SFNA, PSLE, FTNA, CSEE, ACSEE)
- **🎥 Video Hub** — Vipindi vya masomo (Google Drive + GitHub)
- **📖 Kamusi** — Kamusi ya Kiswahili na Kiingereza
- **🔢 Kikokotoo** — Calculator yenye historia
- **👩‍🏫 Teacher Portal** — Walimu wapakie nyenzo zao
- **🏛️ Vyuo Section** — Nyaraka za vyuo vikuu
- **🗺️ Mikoa Resources** — Nyaraka za mikoa mbalimbali
- **🔐 User Dashboard** — Maendeleo, alamisho na notes binafsi
- **🛠️ Admin Panel** — Udhibiti kamili wa maudhui
- **💬 WhatsApp Support** — Msaada wa haraka
- **☁️ Firebase Backend** — Usajili, maudhui ya moja kwa moja na storage

## 🛠️ Teknolojia Zilizotumika

- **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript
- **Backend**: Firebase (Authentication, Realtime Database)
- **Styling**: Custom animations + Glassmorphism design
- **Icons**: Font Awesome 6
- **Hosting**: Static hosting (Netlify / Vercel / GitHub Pages / Firebase Hosting)

## 🚀 Jinsi ya Kuendesha (Local Development)

1. **Pakua au clone** repository
2. **Fungua** `index.html` moja kwa moja kwenye browser
3. **(Optional)** Tumia Live Server (VS Code extension) au `http-server`

**Kumbuka**: Firebase inahitaji configuration yako ili usajili na maudhui ya moja kwa moja yafanye kazi.

## 🔧 Kusanidi Firebase

1. Fungua [Firebase Console](https://console.firebase.google.com)
2. Unda project mpya
3. Washa **Authentication** (Email/Password) na **Realtime Database**
4. Badilisha Firebase config kwenye `index.html` (karibu na mwisho wa `<head>`)
